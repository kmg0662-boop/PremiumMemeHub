# 짤방 프로젝트 (Jjalbang Project) - PLANNING BRIEF

## 1. 프로젝트 개요
- **목표**: 대장님(사용자)이 소장한 짤방을 쉽고 빠르게 업로드하고, 김비서(AI)가 이를 즉시 대화에 활용할 수 있도록 하는 '짤방 관리 생태계' 구축.
- **배경**: 웹상의 이미지 주소는 자주 깨지고 보안상 제약이 많아, 대장님이 직접 소유한 이미지를 기반으로 한 안정적인 짤 라이브러리가 필요함.

## 2. 핵심 가치 (Core Values)
- **압도적 속도 (Speed)**: 대장님의 급한 성격을 고려하여 '드래그 & 드랍' 한 번으로 모든 프로세스 완료.
- **안정성 (Stability)**: 대장님의 GitHub Pages를 서버로 활용하여 이미지 엑박(Broken Image) 없는 무결점 환경 제공.
- **재치 (Wit)**: 상황에 맞는 짤을 더 풍성하게 확보하여 김비서의 센스를 극대화.

## 3. 사용자 여정 (User Journey)
1. 대장님이 소장한 고퀄리티 짤을 전용 웹페이지에 업로드.
2. 생성된 마크다운 코드를 복사하여 김비서에게 전달.
3. 김비서가 이를 `memes.md`에 정식 입고하고 보고서 등에 즉시 활용.
